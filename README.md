To start the node server 

      1.cd backend/
      
      2.npm install
      
      3.node app.js
      (listening at http://localhost:5000)
      
To start the react app

      1.cd restraunt-finder/
      
      2.npm install
      
      3.npm start
      
To run node test cases

      1.cd backend/
      
      2.npm run test
